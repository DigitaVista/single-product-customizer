== Installation ==
Before you install this single product page customizer plugin, make sure you have:

> WordPress 6.9 or newer is installed.
> WooCommerce plugin installed and activated.

1. Go to Plugins > Add New in your WordPress admin.
2. Search for "Single Product Page Customizer with Variation Swatches for WooCommerce".
3. Click "Install Now" and then "Activate".
4. Navigate to WooCommerce > Settings > Single Product Customizer to configure the plugin.
